# Roadmap

## Guiding Principle
Build what proves the loop first. Build what creates lock-in second. Build what drives growth third. Never build V2 features while V1 has unsolved problems.

---

**Goal:** Architecture, design system, and MVP development. Get to a working product before July.

### Product
- [x] Finalize app name — **FOAM** ✅
- [ ] Complete all markdown documentation files
- [ ] Wireframe all MVP screens in UXPilot
- [ ] Design system defined (colors, typography, components)
- [ ] Microcopy finalized for all MVP flows

### Technical
- [ ] Supabase project setup (auth, database schema, storage buckets)
- [ ] Stripe Connect integration for detailer payouts
- [ ] Stripe subscription billing for detailer tiers
- [ ] Core booking engine built and tested

- [ ] App submitted to App Store and Google Play for review

### GTM

- [ ] Founding Detailer offer defined and ready to present

## July 2025 — Soft Launch

**Goal:** First real bookings. Prove transaction loop. Gather signal.

### Product (Must Be Live)
- [ ] Detailer profile setup and service menu builder
- [ ] Customer booking flow end-to-end
- [ ] Real-time availability and calendar management
- [ ] Payment collection, tipping, and payout
- [ ] Automated appointment reminders (push + SMS)
- [ ] Post-job review request flow
- [ ] Before/after photo uploads
- [ ] Crew member basic experience (job view, navigation, completion)
- [ ] Rating and review system

### GTM
- [ ] 15-25 founding detailers onboarded in Atlanta
- [ ] Detailers invite existing customers to book through app
- [ ] Hands-on support for every detailer's first 3 bookings
- [ ] Weekly feedback calls with founding detailer cohort

### Success Criteria to Move to Public Launch
- 100+ bookings completed
- 4.5+ average rating
- Payment success rate 99%+
- Zero critical bugs in core booking and payment flow
- Detailers actively using app to manage their day

---

## August — September 2025 — Public Launch

**Goal:** Open to all Atlanta customers. Begin scaling demand.

### Product
- [ ] App publicly available on App Store and Google Play
- [ ] Referral program (customer refers friend → both get credit)
- [ ] Favorite detailers + notifications when they have openings
- [ ] Saved payment methods and one-tap rebooking
- [ ] In-app messaging between customer and detailer
- [ ] Detailer promo codes and seasonal discounts
- [ ] Recurring appointment setup and automation
- [ ] Waitlist for fully booked detailers

### GTM
- [ ] Paid social campaigns live (Instagram + TikTok, Atlanta geo-targeted)
- [ ] Founder-led organic content publishing weekly
- [ ] Local press outreach (Atlanta Business Chronicle, Axios Atlanta)
- [ ] Referral loop active with founding detailer network

**Goal:** Prove the model is repeatable. Increase revenue per operator.

### Product
- [ ] Market 2 public launch

- [ ] Review response capability for detailers
